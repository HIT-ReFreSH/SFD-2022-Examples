# Markdown

- 轻量级的标记语言
- 书写说明文档、博客都很好用
- 方便排版记笔记
- [$\LaTeX$](https://www.latex-project.org/)格式的公式输入，比大部分输入方式**方便、精准**
- 建议学习$\LaTeX$，写论文的时候比~~Word~~*舒服很多*
